Header

El elemento <header> se utiliza para definir un encabezado para un documento o una sección.

<header>
    <h1>Título</h1>
    <p>Descripción del sitio</p>
</header>

Footer

El elemento <footer> se utiliza para definir un pie de página para un documento o una sección.

<footer>
    <p>Derechos reservados © 2024</p>
</footer>

Color en la Web

Los colores se pueden definir en CSS utilizando nombres, valores hexadecimales o RGB.

.elemento {
    color: #ff0000; /* Rojo */
    background-color: rgb(0, 255, 0); /* Verde */
}
